# Githubs actions has 3 concept

1. Workflows
   1.1 Attached to a github repository
   1.2 Contain one or more Jobs
   1.3 Triggered upon Events
2. Jobs
   2.1 Define a Runner (execution environment)
   2.2 Contain one or more Steps
   2.3 Run in parallel (default) or Sequential
   2.4 Can be conditional
3. Steps
   3.1 Execute a shell script or an Action
   3.2 Can use custom or third-party actions
   3.3 Steps are executed in order
   3.4 Can be conditional
